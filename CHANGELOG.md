# Changelog

All notable changes to VoiceToText will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial monorepo structure combining macOS, Linux, and MLX platforms
- Cross-platform documentation
- Platform comparison guide
- Unified contributing guidelines

## [1.0.0] - 2025-01-27

### Added
- macOS platform support (whisper.cpp + Core ML)
- Linux platform support (FasterWhisper with CPU/GPU)
- MLX platform support (MLX Whisper for Apple Silicon)
- Comprehensive documentation in English and Russian
- Platform-specific README files
- Performance benchmarks and comparisons

### Changed
- Merged three separate repositories into unified monorepo
- Improved documentation structure
- Enhanced cross-platform compatibility
